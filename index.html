I also learnt about CSS(Cascading Style Sheet) which helps to design the layout of a website and it is used alongside with the HTML. There are 3 types of CSS which are inline CSS, Internal CSS and External CSS. I also learnt about the difference between Cascade, Specificity and Inheritance. CSS Box Model includes Margin, Border, Padding, Content. There are also various CSS properties.

Border are used to set the border on an element. There are various CSS border properties which include border-style, border-color, border width and border radius which are used to specify the style, color and size of the border of an element respectively. Padding is the space between the content of an element and the border of an element. we have the top padding, right padding, bottom padding and left padding. Margin are used to define the space around element. it clears an area around the element.
